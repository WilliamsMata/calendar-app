# Calendar.io

**Try it here** -> https://calendario.up.railway.app

### Features

- [x] Create new events

- [x] See other people's events

- [x] Complete calendar

- [x] Edit and Delete your events

- [x] Authentication

- [x] Light and Dark mode

- [x] Tested
